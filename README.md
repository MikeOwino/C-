# CSHARP (C#) .

- int - whole numbers, like: 1, -56, 948

- double - decimal numbers, like: 239.43909, -660.01

- char - single characters, like: “a”, “&”, “£”

- string - string of characters, like: “dog”, “hello world”

- bool - boolean values, like: true or false
